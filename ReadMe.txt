			SPACE INVADERS REMASTERED

Pour lancer le jeu, lancez la commande : $python mainInvaders.py

Un message d'erreur vous disant que le jeu est ouvert dans un fenêtre de terminal trop petite peut apparaitre. Dans ce cas, agrandissez la fenêtre ou mettez la en plein écran appuyant sur F11.

Entrez votre nom comme le jeu vous le demande. Merci de de rentrer que des lettres non accentuées. 

Le jeu va se lancer. Vous êtes donc face à l'écran d'accueil. Appuyer sur "J" pour Jouer, sur "L" pour accéder au tableau des scores, sur "S" pour accéder au réglages, ou sur "ECHAP" pour quitter le jeu.

Lancez la partie en pressant "J", pressez "D" pour faire changer le vaisseau de direction quand vous serez dans le jeu. Vous n'avez pas à vous soucier de tirer, le vaisseau tire en continu.

Quand vous aurez perdu et que vous serez revenu à l'écran d'accueil, vous pouvez presser "L" pour aller voir votre score par rapport aux joueurs précédents. Pressez "A" pour revenir à l'accueil.






ERREURS : 

il se peut que le jeu ne se lance pas et qu'une erreur lié au module json utilisé dans le jeu apparaisse. Dans ce cas, ouvrez le fichier leaderboard.json avec n'importe quel éditeur de texte. Effacer tout le contenu du fichier, écrivez seulement '{}', enregistrer la modification et relancez le jeu. Vous ne devriez plus avoir d'erreurs
